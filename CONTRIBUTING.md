# Contributing

Thanks for your interest.

This repository is published as a frozen release and is not accepting changes.

## No Pull Requests / No Feature Requests

Please do not open pull requests or request new features. This protects the release from drift and keeps the material consistent for readers.

## Forks

You are welcome to fork this repository for personal reference under the license terms. Forks are independent copies and may differ from this release.

## Feedback (Optional)

If you wish to share feedback, use the Calibration materials included in the repository (if present). Sharing feedback is optional and does not imply changes will be made.

Thank you for understanding.
